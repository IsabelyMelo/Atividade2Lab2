# Laboratório 2 - Atividade 2 - Dispositivos Móveis

Aplicativo Android desenvolvido para a disciplina de Dispositivos Móveis que implementa um ForegroundService. O serviço é iniciado por um botão na tela principal e exibe uma notificação persistente com o título "Serviço Ativo" e uma mensagem descritiva.
